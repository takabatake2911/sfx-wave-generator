# SFX Wave Generator

**Retro Sound Effect Generator** — A browser-based app to create retro-style game sound effects.  
You can use presets like Explosion, Laser, Coin, Power Up, Thump, Zap, or customize waveforms, envelopes, and effects.  
Generated sounds can be downloaded as WAV files, and a history feature lets you save up to 30 sounds.

---

## ⚡ Development Setup

```bash
npm install
npm run dev
```
